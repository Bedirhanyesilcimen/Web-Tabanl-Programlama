WorkFlowApp

Projemin konusu çalıştığım şirketteki yönetimsel sürecin dijitalleşmesi üzerine yapılan bir otomasyondur.
Projemin hedef kullanıcı kitlesi Aynen Eldiven çatısında bulunan beyaz yakalı personellerdir ve yönetimdir. Bu otomasyon ile bilgiye ulaşma hızımızı arttırmak ve şirketi içi verilerin daha güvenli ve sağlıklı bir şekilde korunması amaçlanmıştır. 
Şirketimizde bulunan veya yeni katılacak herkes bu otomasyon ile şirketin işleyişine daha çabuk hakim olacak ve daha çabuk bir şekilde uğraşmadan veriye sahip olabilecektir.

Otomasyon içinde kullandığım teknolojiler : C# ASP.NET Core MVC SQL Server ve SQLite (Macbook kullandığım için kendi verilerimi SQLite ile takip ediyorum. Windows olan bir sistemde SQL Server çalışabilecek bir şekilde kodladım.)
Tasarımlar için de Boostrap 5'ten faydalandım.
